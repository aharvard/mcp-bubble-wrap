# MCP Bubble Wrap

A simple MCP (Model Context Protocol) server with HTTP streaming transport.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Production

```bash
pnpm start
```

## Deployment

Configured for [Render.com](https://render.com) deployment via `render.yaml`.
